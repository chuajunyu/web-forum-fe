## Getting Started

Go to `components/constants.ts`, edit the file to point to the correct backend URL

> export const BACKEND_BASE_URL = [BACKEND URL];

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
